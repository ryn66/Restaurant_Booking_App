# Restaurant Hall Booking System

A Flutter-based restaurant hall booking application.

## Features

- User Registration & Login
- Firebase Authentication
- Package Browsing
- Package Search & Filter
- Cart Management
- Booking Management
- Edit Booking
- Cancel Booking
- Firebase Firestore Database
- Admin Management Panel

## Technologies Used

- Flutter
- Firebase Authentication
- Cloud Firestore
- Provider
- Go Router
- Intl
